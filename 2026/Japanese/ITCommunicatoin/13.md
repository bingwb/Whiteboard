## 2. C# <ruby>特有<rt>とくゆう</rt></ruby>の<ruby>機能<rt>きのう</rt></ruby> - C# 特有功能

这些词在 Code Review 中经常出现：

* **<ruby>非同期処理<rt>ひどうきしょり</rt></ruby> (Async/Await):** 异步处理。常说「**非同期（ひどうき）**で<ruby>実装<rt>じっそう</rt></ruby>してください」。
* **<ruby>例外処理<rt>れいがいしょり</rt></ruby> (Exception Handling):** 异常处理 (try-catch)。
* **デリゲート / イベント:** 委托与事件。
* **ラムダ<ruby>式<rt>しき</rt></ruby>:** Lambda 表达式。
* **<ruby>拡張機能<rt>かくちょうきのう</rt></ruby> / <ruby>拡張<rt>かくちょう</rt></ruby>メソッド:** 扩展方法。
* **LINQ (リンク):** * 现场最常用，例如： 「**LINQ（リンク）**で<ruby>抽出<rt>ちゅうしゅつ</rt></ruby>ロジックを<ruby>書<rt>か</rt></ruby>きました」。
* **<ruby>値型<rt>あたいがた</rt></ruby> vs <ruby>参照型<rt>さんしょうがた</rt></ruby>:** 值类型 (struct) 与 引用类型 (class)。
* **ヌル<ruby>許容型<rt>きょようがた</rt></ruby>:** 可空类型 (`int?` 或 `string?`)。

## 3. .NET エコシステム - .NET 生态

* **NuGet (ヌゲット):** 包管理器。常说「**NuGet（ぬげっと）**からライブラリを<ruby>導入<rt>どうにゅう</rt></ruby>する」。
* **ソリューション / プロジェクト:** .NET 的组织结构 (Solution / Project)。
* **<ruby>依存性<rt>いぞんせい</rt></ruby>の<ruby>注入<rt>ちゅうにゅう</rt></ruby> (DI):** Dependency Injection。Azure Functions 或 ASP.NET Core 开发的核心。
* **Entity Framework (EF):** ORM 框架。常说「**イーエフ**でDBアクセスを<ruby>行<rt>おこな</rt></ruby>う」。
* **アセンブリ:** Assembly (编译后的 .dll 文件)。
* **ガベージコレクション (GC):** 垃圾回收。

## 4. <ruby>現場<rt>げんば</rt></ruby>でよく<ruby>使<rt>つか</rt></ruby>われるフレーズ - C# 常用表达

## 5. <ruby>命名規則<rt>めいめいきそく</rt></ruby> - 命名规范术语

日本现场对 C# 的命名习惯非常固守：

* **パスカルケース (PascalCase):** 类名、方法名。
* **キャメルケース (camelCase):** 局部变量。
* **スネークケース (snake_case):** 日本现场一般**不建议**在 C# 中使用，除非是 DB 映射。
* **ハンガリアン<ruby>記法<rt>きほう</rt></ruby>:** 匈牙利命名法（虽然现在不流行，但在一些老旧系统现场可能还会见到）。

* **デバッグ<ruby>実行<rt>じっこう</rt></ruby>:** 调试运行。
* **ブレークポイントを<ruby>張<rt>は</rt></ruby>る:** 设断点（这里动词也用「<ruby>張<rt>は</rt></ruby>る」）。
* **ステップ<ruby>実行<rt>じっこう</rt></ruby>:** 单步执行。
* **ウォッチ<ruby>式<rt>しき</rt></ruby>:** 监视窗口/表达式。
* **スタックトレース:** 堆栈轨迹。
